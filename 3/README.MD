## 1. Мета роботи

Навчитися вимірювати основні характеристики операційного підсилювача: коефіцієнт підсилення при інвертуючому та неінвертуючому включенні, власний коефіцієнт підсилення, коефіцієнт пригнічення синфазної завади.

## 2. Обладнання
* Програмне середовище **NI Multisim**

## 3. Хід роботи

### Визначення власного коефіцієнта підсилення операційного підсилювача

Оскільки пряме вимірювання власного коефіцієнта підсилення вимагає подачі на вхід ОП надмалих напруг (порядку мікровольтів), що складно реалізувати на практиці через шуми, було зібрано спеціальну схему із "занулюючим" підсилювачем. Ця схема дозволяє вимірювати коефіцієнт підсилення ОП опосередковано.

Для дослідження було використано мікросхему **TL082CP**.

Отримані значення вихідної напруги:
* Для $V_{test} = -1\text{ В}$:  $V_{out1} = -16.015\text{ мВ}$
* Для $V_{test} = +1\text{ В}$:  $V_{out2} = -5.949\text{ мВ}$

#### Аналіз та розрахунок характеристик

1. Розрахуємо різницю напруг на виході "занулюючого" підсилювача $\Delta V_{o-null}$:
   $$\Delta V_{o-null} = V_{out2} - V_{out1}$$
   $$\Delta V_{o-null} = -5.949\text{ мВ} - (-16.015\text{ мВ}) = 10.066\text{ мВ}$$

2. Визначимо різницю напруг, що з'являлася безпосередньо на вході досліджуваного підсилювача $\Delta V_{in-dut}$ (через дільник напруги $R_1$ та $R_2$):
   $$\Delta V_{in-dut} = \frac{R_1}{R_1 + R_2} \cdot \Delta V_{o-null}$$
   $$\Delta V_{in-dut} = \frac{100}{100 + 100000} \cdot 10.066\text{ мВ} = 0.010056\text{ мВ}$$

3. Розрахуємо власний коефіцієнт підсилення досліджуваного ОП ($G_{ol}$). Знаючи, що зміна вихідної напруги тестового підсилювача становила $\Delta V_{o-dut} = 2\text{ В}$ (від $-1\text{ В}$ до $+1\text{ В}$):
   $$G_{ol} = \frac{\Delta V_{o-dut}}{\Delta V_{in-dut}}$$
   $$G_{ol} = \frac{2\text{ В}}{0.010056\text{ мВ}} \approx 198.89\text{ В/мВ}$$

## 4. Висновки

Під час виконання практичної роботи №3 було вивчено методики вимірювання основних характеристик операційного підсилювача (ОП). Основний акцент було зроблено на експериментальному визначенні власного коефіцієнта підсилення без розімкненого зворотного зв'язку ($G_{ol}$), оскільки пряме вимірювання цього параметра ускладнене необхідністю генерувати та вимірювати сигнали мікровольтового діапазону.

Для обходу цієї проблеми було успішно змодельовано схему із застосуванням допоміжного "занулюючого" підсилювача та високоомного дільника напруги. На базі операційного підсилювача TL082CP було виміряно макроскопічні зміни напруги та перераховано їх у мікровольтові відхилення на входах. 

Експериментально розрахований власний коефіцієнт підсилення склав $G_{ol} \approx 198.89\text{ В/мВ}$. Це значення з високою точністю корелює із заявленими характеристиками виробника (згідно з технічною документацією Texas Instruments, типове значення *Large-signal differential voltage amplification* для мікросхем серії TL082 становить $200\text{ В/мВ}$). Це підтверджує надійність застосованого вимірювального методу та правильність зібраної моделі.
